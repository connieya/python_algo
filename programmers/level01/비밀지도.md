# 프로그래머스 Lv.1 비밀 지도

## solution 1

```python
def solution(n, arr1, arr2):
    answer = []
    for i in range(n):
        arr1[i] = arr1[i] | arr2[i]
    
    for i in range(n):
        num = arr1[i]
        str = ""
        while num > 0:
            str += "#" if num%2 == 1 else " "
            num //= 2
        while len(str) < n:
            str += " "
        answer.append(str[::-1])
    return answer
```

# 프로그래머스 Lv.1 둘만의 암호

## solution 1 아스키코드 , 구현

```python
def solution(s, skip, index):
    answer = ''
    for elem in s:
        cnt = index
        while cnt > 0:
            n = chr(ord(elem)+1)
            if n >'z' : n='a'
            while n in skip:
                n = chr(ord(n)+1)
                if n >'z':
                    n = 'a'
            
            cnt -=1
            elem = n
        
        answer += elem
    return answer
```

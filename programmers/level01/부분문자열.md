# 프로그래머스 Lv.1 크기가 작은 부분 문자열

## solution 1

```python
def solution(t, p):
    answer = 0
    sz = len(p)
    for idx in range(len(t)-sz+1):
        i = 0
        st = ""
        while i < sz:
            st += t[idx+i]
            i +=1
        if int(st) <= int(p):
            answer += 1
    return answer
```

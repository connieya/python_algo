# 프로그래머스 Lv.3 가장 긴 팰린드롬

## solution 1

```python
def solution(s):
    answer = 1
    for idx in range(len(s)):
        for j in range(len(s),idx+1,-1):
            v1 = s[idx:j]
            v2 = v1[::-1]
            if v1 == v2:
                answer = max(answer, j-idx)
            
    return answer
```

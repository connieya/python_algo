# 프로그래머스 Lv.3 풍선 터트리기

## solution 1 Stack

```python
def solution(a):
    answer = len(a)
    stack = []
    for num in a:
        while stack and stack[-1] > num :
            stack.pop()
            if stack:
                answer -= 1 
        stack.append(num)
            
    return answer
```

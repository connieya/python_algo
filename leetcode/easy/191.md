# LeetCode 191. Number of 1 Bits

## solution 1 bin

```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        return bin(n).count('1')
```

class Solution:
    def hammingWeight(self, n: int) -> int:
        count = 0 
        while n:
            n &= n-1
            count +=1
        return count

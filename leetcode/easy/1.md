# LeetCode 1. Two Sum

## solution 1

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        dict = {}
        for i in range(len(nums)):
            if nums[i] in dict:
                return [ dict[nums[i]] , i]

            dict[target-nums[i]]  = i
        
        
```

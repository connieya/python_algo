# LeetCode. Subsets

## solution 1

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        ans = []
        def dfs(depth , arr):
            if depth == len(nums):
                ans.append(arr)
                return
            
            dfs(depth+1 , arr+[nums[depth]])
            dfs(depth+1 , arr)
            
        dfs(0,[])
        return ans
```

## solution 1 


```python
import sys
month = [0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30]
weekend = ["SUN", "MON", "TUE", "WED", "THU", "FRI", "SAT"]
m, d = map(int, sys.stdin.readline().split())
days = 0
for i in range(m):
    days += month[i]

days += d
print(weekend[days % 7])

```

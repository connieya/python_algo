## solution 1

```python
n,m = map(int,input().split())
visited = [False]*(n+1)

def dfs(depth , arr):
    if depth == m:
        for a in arr:
            print(a,end=' ')
        print()
        return
    for i in range(1,n+1):
        if not visited[i]:
            visited[i] = True
            dfs(depth+1,arr+[i])
            visited[i] = False

dfs(0,[])
```

## solution 2

```python
from itertools import permutations

n,m = map(int,input().split())
arr = map(str,range(1,n+1))
print('\n'.join(list(map(' '.join, permutations(arr,m)))))
```
